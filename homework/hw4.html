Answers to questions

  1. The box model states that all HTML elements can be considered as boxes. Each element is wrapped by box with consist of margins, borders, padding, and content.
  2. A "block-level" element occupies the entire space of its parent element. They always start on a new line and takes up the full width of parent element.
  3. Paragraph is an example tag that is by default rendered as block-level element.
  4. An "inline" element occupies the space bounded by the tags defining the element. They do not start on a new line and only takes up as much width as necessary.
  5. Button is an example tag that is by default rendered as inline element.
  6. CSS can be included in a seperate file and linked to in html file using link tag, or it can be embedded into HTML file using style tag inside head.
  7. CSS selectors are patterns used to select elements for styling.
  8. The following CSS rule can be used to set the background color of body to #0000FF (blue).body { background-color: #0000FF; }
